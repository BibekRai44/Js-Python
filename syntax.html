Syntax Differences between Javascript and Python

Declaration of syntax

Syntax Python Javascript
Comments #This is comment in Python. //This is comment in Javascript.
Variables a=5 , b="Bibek" let a=5 , var a=5 , const a=5
Data Types a=5 #int , a="bibek" #string let a=5 //number , var a="bibek" //string
Numbers a=1 #int, a=1.3 #float , a=1b #complex let a=1 //A number without decimal , let a=1.3 //A number with decimal
Strings a="hello" or a='hello' #Both are accepted. let a="hello" or let a='hello' //Both are accepted.
Boolean print(1>2) = False #python takes only True or False console.log(1>2) = False #Javascript also takes the same as python.
